使用Webpack CSS loader加载器

  • 1.什么是加载器
  • loader用于转换应用程序的资源文件，他们是运行在nodejs下的函数，使用参数来获取一个资源的来源并且返回一个新的来源（资源的位置）
  • 2.loader的解析
  • loader的解析类似模块，一个loader模块会导出一个方法并且可被nodejs写为可兼容的JavaScript,通常情况下通过npm来管理loader,但你也可以把loader放在自己的应用里。
  • 3.安装loader

  • 你可以通过npm来安装loader $ npm install xxx-loader --save-dev 或者 $ npm install xxx-loader --save
